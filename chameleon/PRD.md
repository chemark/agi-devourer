# 游戏 PRD 文档：《AGI 吞噬者》（原名：打嗝变色龙）

## 1. 游戏概述
**标语 (Pitch)：** “你只能做一件事：按下空格，吞噬大模型，进化为真正的 AGI！”
**背景故事：** 一只背上印着 “AGI” 的终极赛博变色龙，想要突破算力奇点成为真正的数字神明。玩家需要控制它不断吐出变异的舌头，将天空中代表各大主流 AI 语言模型的“胶囊飞虫”吞入腹中进行融合。
**核心机制：** 单按键（空格/点击）街机反应游戏。极简的交互结合基于真实大模型（国内全域直连：MiniMax-M2.5）API 生成的动态反馈，给玩家带来充满荒诞科技圈黑话的无限“盲盒”喜剧体验。

## 2. 核心玩法 (Core Gameplay Loop)
1. **自动挂机瞄准（限制挑战）：** 变色龙的本能导致舌头瞄准虚线会像钟摆一样在上方（-45° 到 +45°）恒定来回匀速平移（速度 1.2，容错极高）。
2. **唯一交互输入（One Thing 原则）：** 全程玩家唯一的合法操作是：**看准时机，敲击空格键（或鼠标左键）**。按下后，舌头将沿着当前的瞄准轨道瞬间飞出。
3. **抓取与收网（宽幅碰撞）：** 将原本严格的坐标点重构成了“包含舌头红球半径的 +25px 宽幅擦边判定”。若击中了正在乱飞的大模型实体，会将其卷入口中，瞬间斩获其自带的基础算力值（例如 OpenAI: 300分，Kimi: 200分）。
4. **终极算力融合（大招释放）：** 累积吞噬满 **3** 个大模型后，胃部算力过载，强制接管并触发最核心的“大模型化学变异结算”。

## 3. API 叙事系统 (AI Innovation - 30% 权重)
作为占据评审 30% 创新比重的最大化抓手，我们彻底换下了必须依赖出海隧道的原始底层，直接切换接驳至具备超低无响应率的 **阿里云 DashScope (集成 MiniMax)**，通过标准的跨源格式进行稳定提取。
* **输入组合变量：** 传给 AI 的是这极短时间内被生吞的 3 家异构大语言模型的名字及其乱炖混战。
* **人格面具注入：** 极具幽默感的癫狂科技圈解说员。
* **输出结果：** 动态返回 1 句极限字数压缩的“算力碰撞事故小作文”（充满各种 IT 行业黑话或离谱的肉体赛博化变异），以及一个根据模型互斥程度给出的融合附加战力分（10~100）。
* **极端网络容灾设计：** 如果在内网路演或遭代理切断，触发捕获后系统将静默介入长达 5 套的随机“离线专属梗备用笑话库”（并且保底抽取 50~100 补充分数），保证游戏心流的绝对连贯，永不存在红字卡死问题。

## 4. 胜负体验规划 (Win & Lose Mechanism)
* **全局时限引擎：** 玩家拥有一段极短且无增量获取途径的 **120 秒** 倒计时压力阀。
* **赢者加冕：** 基础击杀分结合每次吃撑结算的 Gemini 爆分，一旦总额越过 **4000 算力临界区**，系统将瞬间强制锁定一切活动并高能宣布：“算力融合已达到临界值！变色龙突破奇点，进化为真正的 AGI！”
* **失败机制：** 如果手脚迟缓满场溜怪，致使 120 秒内依然在 4000 分大关外徘徊，将直接落入失败判定：“算力枯竭！在降智幻觉中停止了思考！”
* **无缝轮回的内存流机制：** 每局无论死活，均在 UI 高亮提供丝滑纯洁的纯逻辑数据劫持清空按钮“重新开始吞噬”，完成内存级变量零延迟清洗和开机。

## 5. 美术与音效 (Art & Audio)
* **2.5D 美术景深：** 高饱和全屏框架 (100vw/vh)。采用 AI 渲染完成的脱底天空辅以带有 CSS `filter` 对比亮色洗底魔法的底层变色龙模型，叠加极高文字排布占比的 AGI 渐变呼吸霓虹流光。将原来的靶形元素全面废除并进化为了胶囊宽横向条带展示的大语言模型的长文本牌匾。
* **纯代码轻巧声学结构：** 0 外挂 MP3 依赖。纯靠代码实例化浏览器的 AudioContext，榨取出最硬核无延时的 **8-bit** 频响器发声。含有：
    1. “嗖”：通过 `triangle` 三角波实现频率光速爬坡的弹射音。
    2. “吧唧”：由正弦波交叠形成的进食两极脆拍。
    3. “嗡——”：作为最终结算反馈音，基于底层锯齿波并附加癫狂的 `detune` 音高乱颤错位，硬生生合成出了“深腹空鸣”般荒加打颤的废料长嗝低音炮效果，完美契合魔幻叙事主题。
