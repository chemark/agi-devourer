# 舌尖上的 AGI V2 Alpha

当前分支正在重建 V2 Alpha，开发入口以 [`chameleon-v2/`](./chameleon-v2) 为准。

## 当前目标

- 竖屏产品首页
- 横屏准备页
- 横屏 60 秒挑战
- 结算页
- mock 微信登录
- mock 榜单反馈

## 本地启动

```bash
./start.sh
```

或手动运行：

```bash
cd chameleon-v2
npm install
npm run dev -- --host 127.0.0.1 --port 4173
```

默认地址：

```text
http://127.0.0.1:4173
```
