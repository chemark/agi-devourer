# 🗺️ 《舌尖上的 AGI》版本迭代与开发路线图 (Roadmap)

本文档用于跟踪游戏从 GameJam 原型阶段迈向完整线上版本的后续功能开发进度。
在此记录所有的需求池及任务拆解，每完成一项即可勾选。

---

## 阶段一：建立账号与荣誉竞技体系

- [ ] **1. 得分排行榜与用户注册**
  - [ ] **账号体系基础**：搭建并实现用户注册和登录体系。
  - [ ] **多种登录方式**：打通第三方登录授权，支持微信、手机号短信验证码以及邮箱注册。
  - [ ] **双轨游玩模式**：支持“游客模式”（无需注册门槛即可随时体验游戏），但游客积分不上榜；如果希望成绩进入全服排行榜，则系统必须要引导并强制游客注册/登录账号。
  - [ ] **全球排行榜 (Leaderboard)**：按最高得分进行排序，记录并展示排名前列的玩家。

---

## 阶段二：打通移动端与社交红利

- [ ] **2. 移动端体验优化**
  - [ ] **屏幕尺寸与布局适配**：针对手机的竖屏或横屏体验优化 Canvas 的缩放机制，确保所有提示、UI 按钮在小屏幕上也不变形。
  - [ ] **手势与触控映射**：由于原游戏依赖鼠标瞄准+空格发射，需重写底层控制逻辑，实现手机上的触控屏幕滑动瞄准，点击屏幕空白处或虚拟按钮发射舌头，保证流畅的手感。
  
- [ ] **3. 社交分享与裂变传播**
  - [ ] **社交模块按钮接入**：在紧张刺激的游戏 Game Over 结算页面，增加显眼的“分享”和“炫耀”按钮。
  - [ ] **自动化战报海报生成**：基于 `html2canvas` 等技术，在结算画面自动生成带有玩家专属“算力怪嗝”文案、最终奇点得分以及游戏引流二维码的精美战报海报，方便玩家一键发朋友圈或微信群进行社交裂变传播。

---

## 阶段三：硬核玩法与难度曲线升级

- [ ] **4. 动态游戏难度系统重设**
  - [ ] **随着时间推移的整体加速系统**：目前的 Demo 版本过于简单护肝，需要实现一个全局计时器变量，让游戏的整体运行速度、重力或物理系数随着生存倒计时时间的不断延长变得越来越鬼畜和疯狂。
  - [ ] **模型实休捕获的 AI 闪避逻辑**：调高舌头捕获的命中判定难度。天上飞的这些大模型（猎物）不应再只是简单的直线或抛物线呆板运动，需要在后期推进阶段赋予它们更诡异的曲线、弹跳、甚至是躲避玩家舌头准星的高级轨迹，让“打准”变为极限挑战！
